# Security Policy

## Supported Versions

We only support the latest deployment on Hugging Face Spaces.

| Version | Supported |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability (e.g., API Key leakage, SQL Injection risk), please **DO NOT** open a public issue.

1.  Email me directly at: **tosinowadokun11@gmail.com**
2.  I will acknowledge your email within 48 hours.
3.  I will provide a fix timeline.

Thank you for responsible disclosure.
